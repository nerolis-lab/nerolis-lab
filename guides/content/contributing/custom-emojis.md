Guide pages can use shortcodes to include small inline images (berries, ingredients, emotes, etc.). These work like custom emoji on Discord: emoji name wrapped in colons (i.e. `:greengrass:`);

## :notes: Usage

Examples:

- `:belue:`: Belue berry image from `images/emojis/berries/belue.png`
- `:sad:`: Sad mascot image from `images/emojis/emotes/sad.png`

Shortcodes only match **known** names. Anything that is not a registered file is left as plain text.

Emojis scale inline with the surrounding text. For example, notice this one is smaller than the one in the section header. :notes:
