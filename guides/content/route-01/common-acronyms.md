## General

| Term    | Definition                                                                                     |
| ------- | ---------------------------------------------------------------------------------------------- |
| Pips    | Points required to befriend a Pokémon                                                          |
| FL      | Friendship Level                                                                               |
| RP      | Research Power                                                                                 |
| DP      | Drowsy Power                                                                                   |
| AB      | Area Bonus                                                                                     |
| Ing     | Ingredients :ingredients:                                                                      |
| SS      | Sneaky Snacking (see [Sneaky Snacking Explained](../introductory-concepts/sneaky-snacking.md)) |
| A/B/C/X | Ingredient order/slot e.g. AAA, ABB, ABC, AAX                                                  |
| Potmon  | Pokémon with `Cooking Power-Up S` main skill                                                   |
| Proc    | Skill activation :skillproc:                                                                   |
| Trigger | Any of STS/STM or MSC+ nature                                                                  |
| Ribbons | Accrued sleep time bonuses :ribbon4:                                                           |
